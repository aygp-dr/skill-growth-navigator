# skill-growth-navigator

Learning path recommendation engine. Built with Flask, deployable on Replit.

## Run locally

```bash
pip install -r requirements.txt
python main.py
```

## Deploy on Replit

Fork this repo on Replit — it auto-detects the `.replit` config.

## Stack

- Python 3.11+ / Flask
- SQLite (in `data/`)
- Vanilla CSS (dark theme)
