#+TITLE: skill-growth-navigator
#+STARTUP: overview

* Overview

Learning path recommendation engine. Built with Flask, deployable on Replit.

* Quick Start

#+begin_src bash
make setup
make run
# Open http://localhost:8080
#+end_src

* Development

| Target | Description |
|--------+-------------|
| make setup | Create venv, install deps |
| make run | Start Flask server |
| make test | Run pytest |
| make lint | Run ruff linter |
| make clean | Clean caches and db |

* Deploy on Replit

Fork this repo on Replit — it auto-detects the =.replit= config.
